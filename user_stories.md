# User Stories — RemindMe App

## User Story 1 — User Registration
As a new user, I want to create an account with a username, email, and password,
so that I can access the app securely and save my personal reminders.

## User Story 2 — User Login
As a registered user, I want to log in with my email and password,
so that I can access my reminders from any session.

## User Story 3 — Create a Reminder
As a logged-in user, I want to add a reminder with a title, description, and date/time,
so that I never forget important tasks or events.

## User Story 4 — View Reminders on Home Screen
As a user, I want to see all my reminders listed on the home screen,
so that I can quickly review what I need to do.

## User Story 5 — View Reminder Details
As a user, I want to tap a reminder to see its full details,
so that I can read the complete information about that reminder.

## User Story 6 — Persist Reminders Locally
As a user, I want my reminders to be saved on my device,
so that they are still available after I close and reopen the app.

## User Story 7 — Fetch Inspirational Quotes via API
As a user, I want to see a motivational quote on my home screen fetched from an external API,
so that I feel inspired when managing my tasks.

## User Story 8 — Receive Push Notifications
As a user, I want to receive a push notification when a reminder is due,
so that I am alerted even when the app is in the background.

## User Story 9 — Customize App Settings
As a user, I want to access a settings screen where I can toggle dark mode,
manage notification preferences, and clear all my data,
so that I can personalize my app experience.
